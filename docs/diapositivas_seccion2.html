2.1 — Instalación

¿Cómo se instala Node-RED?

  • 💻
    npm (local) — requiere Node.js instalado. Ideal para desarrollo individual.
    npm install -g --unsafe-perm node-red
  • 🍓
    Raspberry Pi — script oficial que configura Node.js y Node-RED automáticamente.
  • 🐳
    Docker ★ Método del equipo — imagen reproducible, mismo entorno en cualquier SO.
# Levantar el entorno completo en un solo comando
docker compose up --build -d
Ventaja clave: cualquier integrante clona el repo y levanta el entorno idéntico — sin instalar Node.js manualmente.

2.4 — Categorías de Controles y Funciones

¿Qué nodos existen?

⚙ Common
inject · debug · catch · complete · comment
𝑓 Function
function · switch · change · template · json · delay
🌐 Network
http in · http response · http request · mqtt · websocket
💾 Storage
file · file in · watch
📊 Dashboard (node-red-dashboard)
ui_button · ui_form · ui_table · ui_chart · ui_toast · ui_text · ui_tab · ui_group
El DSL visual de Node-RED permite construir aplicaciones completas combinando estas categorías — sin escribir código en la mayoría de los casos.
